digitalmarketplace-api
======================

API tier for Digital Marketplace.

- Python app, based on the [Flask framework](http://flask.pocoo.org/)

### Setup (mac)
#### Get virtualenv
	sudo easy_install virtualenv

#### Set up env
	virtualenv venv

#### Activate
	source venv/bin/activate

#### install flask
	pip install flask

#### run it
	./application.py runserver
